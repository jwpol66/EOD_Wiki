# EOD 장비 (EOD Equipment) 🤖

현장에서 폭발물 의심 물체를 정찰하고 판독하며, 안전하게 무력화하기 위해 운용하는 핵심 장비들입니다. 각 장비의 제원과 한계를 명확히 숙지하고 적재적소에 활용하는 것은 작전의 성공과 직결됩니다.

  <a href="#/equipment/xray" class="case-card">
    <h4>☢️ X-ray 장비</h4>

  </a>

  <a href="#/equipment/robot" class="case-card">

    <p>원격 조종을 통해 의심 물체에 접근하여 정찰, 인양, 처리 등의 임무를 수행하는 다목적 무인 로봇(SPUR)입니다. 요원의 위험 노출을 획기적으로 줄여줍니다.</p>
  </a>

  <a href="#/equipment/eod10e" class="case-card">

  </a>

  </a>

  </a>

  <a href="#/equipment/hook_n_line" class="case-card">
    <h4>🪝 이동봉 & 훅앤라인</h4>

  </a>

<br>

> **⚠️ EOD 현장 안전 수칙:**
> 모든 장비는 출동 전 철저한 배터리 및 작동 점검이 필수입니다. 현장 투입 시에는 로봇과 X-ray 등 원격 정찰 장비를 최우선으로 투입하여 요원의 직접 접근(Manual Approach)을 최소화해야 합니다.
