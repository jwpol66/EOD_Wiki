# 🧨 폭약 (Explosives)

EOD 작전 및 훈련 시 참고하는 주요 폭약의 제원과 특성을 정리한 카테고리입니다. 
아래 항목을 클릭하면 상세 제원을 확인할 수 있습니다.

---

### 📂 세부 항목 리스트

1. **[TNT (Trinitrotoluene)](explosives/tnt.md)**
   * 모든 폭약 위력 측정의 **표준(Standard)**이 되는 고체 폭약.
   * 높은 안정성과 주조(Casting) 가능한 특성 보유.

2. **[C4 (Composition C-4)](explosives/c4.md)**
   * RDX 주성분의 대표적인 **가소성(Plastic)** 폭약.
   * 손쉬운 성형성과 수중 사용 가능 기능 보유.
